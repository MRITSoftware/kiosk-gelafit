# Sistema de Geração de Orçamentos Barrella

Este sistema permite criar e editar PDFs de orçamentos usando interface Tkinter e ReportLab.

## Estrutura
- `app.py`: Início do programa
- `gui/`: Janelas Tkinter (principal, opções, contatos, edição)
- `pdf/`: Funções de geração e utilitários para PDF
- `services/`: Comunicação com a API externa
- `data/`: Manipulação de dados locais (contato)
- `assets/`: Ícones, imagens e recursos estáticos
