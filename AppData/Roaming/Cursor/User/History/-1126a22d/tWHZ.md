# Pasta de Imagens

Esta pasta é destinada para armazenar as imagens da landing page.

## Imagens Necessárias

### Hero Section (Carrossel)
- `hero-1.jpg` - Imagem principal com palco e LED screens
- `hero-2.jpg` - Imagem de equipamentos de última geração
- `hero-3.jpg` - Imagem da estrutura de 1500m²

### Sobre Nós
- `sobre-warehouse.jpg` - Imagem do armazém de 1500m²

### Vídeos
- `video-1.jpg` - Thumbnail do vídeo de show musical
- `video-2.jpg` - Thumbnail do vídeo de evento corporativo
- `video-3.jpg` - Thumbnail do vídeo de festival
- `video-4.jpg` - Thumbnail do vídeo de lançamento de produto

### Clientes
- `cliente-netflix.jpg` - Logo/evento Netflix
- `cliente-budweiser.jpg` - Logo/evento Budweiser
- `cliente-moet.jpg` - Logo/evento Moet & Chandon
- `cliente-vtex.jpg` - Logo/evento VTEX Day
- `cliente-comgos.jpg` - Logo/evento Comgos

## Como Adicionar Imagens

1. Adicione as imagens nesta pasta
2. Atualize o HTML substituindo os placeholders pelas imagens reais
3. Adicione as classes CSS apropriadas para lazy loading se necessário

## Especificações Recomendadas

- **Formato**: JPG ou PNG
- **Resolução**: Mínimo 1920x1080 para imagens de fundo
- **Tamanho**: Máximo 500KB por imagem para melhor performance
- **Otimização**: Use ferramentas como TinyPNG para comprimir as imagens
