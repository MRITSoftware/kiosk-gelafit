# GelaFit - Sistema de Controle de Produtos

O GelaFit é um sistema de controle de produtos desenvolvido para a GelaFit pela empresa MRIT Software. Ele permite ao proprietário da loja cadastrar produtos, controlar o estoque, gerar relatórios de vendas e realizar ações relacionadas ao gerenciamento de produtos.

## Funcionalidades Principais
Cadastro de produtos com informações detalhadas.
Visualização da lista de produtos cadastrados.
Geração de relatórios de vendas.
Controle de estoque, com alertas de baixo estoque.
Funcionalidade de venda de produtos, atualizando automaticamente o estoque.

# Como Usar
Clone o repositório para o seu ambiente local.

git clone URL_DO_SEU_REPOSITORIO_GIT
Compile e execute o programa Java.

javac Main.java
java Main
